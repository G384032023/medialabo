「Webコンテンツ作成」の成果

本テーマについて

情報工学の専門科目の基礎知識を応用して実践的な能力を習得することが目的．目標は4つ．1.実験の一連の流れを理解して実践できる．2.グループワークを円滑に実施できるようになる，3.モーション解析と編集ができる，4.HTML等を用いたWebページの作成ができる．

プロジェクト課題について

WWWのページへ世界の天気を検索して検索結果を表示する．選択された国とその国の選ばれた項目に関する情報を表示する．ユーザーは国を選択した後に，天気，気温，湿度，風速などの8つの項目の中から自分の知りたい情報だけを選んで知ることができる．しかし，国だけを選んで項目を何も選択しないとその旨を伝えるメッセージが表示されるようにプログラミングを組んだ．

プロジェクト課題で作成したページ:project.html

数当てゲーム

ユーザーはランダムに設定された数を予想して，テキストボックスに最大で3回まで入力する．不正解だった場合，入力された数より大きいか小さいかを表示する．3回以内に正解出来ればユーザーの勝利，そうでなければ負け．入力バーに入力された数字を反映して，それが何回目の回答なのか， 答えより大きいのか小さいのかをその都度更新するプログラミングを組んだ．また，4回目以降は答えを表示し，すでにゲームは終わっていることを伝えるプログラミングも組んだ．入力された数字は文字列として認識されるためそれを数字と認識させなおすプログラミングを組むのが大変だった．

演習の成果:数当てゲーム

各授業回の学習事項

第1回 WWW と HTML

HTMLを使って簡易的なページを作りました．今日習ったことは一通りは1年生の時に少しだけやった気がしますが，記憶があまりなかったためほぼ初見でした．でも思ったよりは簡単だったので次回以降も頑張っていきたいです．

演習の成果: 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

CSSを使ってページの装飾をしました．今回のところも中学時代に習いましたが，色のグラデーションや文字に影をつけるなどの細かいところは初めてやりました．画像の位置の調整や，文字を入れるボックスなど知らない要素がたくさんあったので復習をしっかりしたいです．

演習の成果:練習問題2-3

第3回 JavaScript の基本

JavaScriptを初めて使いました．Javaを使っていた頃の癖が色々出てしまってまだ慣れないし切り替えが難しいですが，慣れていけるようにしたいです．それとVSコードに試しで書いたコードを貼り忘れないように注意したいです．

演習の成果:練習問題3-6

第4回 DOM と JavaScript

DOMを使ってHTMLを使わずにページの内容を変更しました．今回は考えなければならないことが多く，今までで一番難しいと感じました．また前回と同じようにVSコードに直接書き込んでしまったことが何回かあったのでそろそろ慣れていきたい．

演習の成果:練習問題4-2から4-5

第5回 イベント処理

前回までのJavaScriptやDOMより複雑ではなくとても簡単で楽しく勉強できました．でも，キーボードを押した時や押して離した時などかなり細かい設定をするところがあってこんがらがらないか心配になりました．

演習の成果:練習問題5-3

感想など

JavaScriptもDOMも初めて使ったのでとても難しかったです．HTMLとCSSを使ってWebサイトを作ったことはありましたがJavaScriptやDOMを使って検索バーなどを作成してそれをプログラムで動かすのは初めてだったので全然わかりませんでした． さらに，DOMを使ってHTMLを書き換えたりするのも今まで経験してきたプログラミングとは全くやり方やコードが異なっていて，とても難しく，専用のコードを使いこなすのはとても難しかったです．また，Javaを使っていた時の癖がなかなか抜けず， すぐにVScodeに直接書いてしまっていたのでそこも慣れるのが大変でした．でも，いざちゃんと思った通りにプログラムが動くととても達成感があり，同時に楽しさも感じました．プロジェクト課題の，項目が選ばれていない時のエラーメッセージを出すif文が なかなか正常に動かず，すごく悩みましたが，いざ原因が見つかってそれにしっかり対処してちゃんとエラーメッセージが表示された時はとても嬉しく感じました．今思えばもうちょっとちゃんと考えていればすぐにわかったことが原因だったのですが， それを考える余裕がないほど他のプログラムやCSSを考えたりオリジナリティの出し方を考えるのが大変で，マルチタスクが苦手な私はすごく苦労しました．その分ちゃんと動いた時の感動や，思い通りの構図になった時の嬉しさや達成感は大きいのですが， 疲れの方が多く，まだまだ楽しんでこの作業をするには道のりが長そうだなと思いました．それらに比べて，イベント処理の部分は理解しやすく，すんなりと頭に入ってきたし，楽しんで学ぶことができました．
